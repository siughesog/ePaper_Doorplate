# 智慧電子紙門牌系統 - 專題評分報告

## 📋 專案概述

**專案名稱**: 智慧電子紙門牌系統 (Smart ePaper Doorplate System)  
**技術棧**: 
- 後端: Spring Boot 3.4.0 (Java) + MongoDB
- 前端: React 19.1.0 + Tailwind CSS
- 硬體: ESP32 + GxEPD2 ePaper 顯示器
- 部署: Railway (後端) + Vercel (前端)

---
